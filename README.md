# Semiconductor Industry Intelligence Dashboard

An interactive dashboard analysing the global semiconductor industry 
across 4 dimensions using 5 real-world datasets from Kaggle.

## Datasets Used
- AI Chip Market (120 rows) — vendor revenue, shipments, performance
- Company Financials (617 rows) — revenue, margins, R&D for 40 companies
- Chip Prices (405 rows) — DRAM, NAND, HBM, NVIDIA H100 and B200 pricing
- Export Controls (34 rows) — US-China geopolitical actions and severity
- Fab Capacity (313 rows) — wafer capacity by region and process node

## Key Insights
- NVIDIA dominates AI chip revenue with 59.5% operating margin in 2024
- Taiwan (TSMC) accounts for majority of global leading-edge fab capacity
- Biden administration issued the most severe export controls (severity 10)
- GPU prices (H100 at $35,640) are 5,000x higher than memory chip prices
